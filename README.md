# LAVACore
Convective Impulse Heating simultaneously compatible with torch-flame &amp; magnetic induction.

N.B.:  Ayant reconnu l’antériorité du brevet VaporGenie de Dan Steinberg (2003), j’offre ici une extension conceptuelle surnommée « LAVACore ».  Cette contribution a été conçue dans un esprit d’ouverture, sans prétendre disputer des droits de propriété intellectuelle pré‑existants.  La documentation de ce projet modulaire dans le domaine public vise à favoriser la liberté d’innovation pour tous et à inscrire ma démarche dans une continuité technique évolutive.  Cette contextualisation a également pour but d’éviter toute interprétation pouvant être comprise comme une prise de position juridique.
